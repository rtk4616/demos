News - kramdown Online Editor / kramup

A simple online editor that lets you try the kramdown library converting markdown to HTML (or even LaTeX ;-)). The online app also includes a HTTP JSON API service (e.g. /markdown) for conversion. All is packed up in the kramdown-service gem for easy (re)use and as an extra bonus includes a binary, that is, kramup, that starts up the online editor and service on your local machine (e.g. use $ kramup). Supported conversion options/modes include:

  • HTML - kramdown in GitHub-Flavored Markdown (GFM) Mode
  • HTML - kramdown in GitHub-Flavored Markdown (GFM) Mode w/ Syntax Highlighter (rouge)
  • HTML - kramdown in “Classic” Mode
  • LaTeX - kramdown in GitHub-Flavored Markdown (GFM) Mode

All code public domain.

News - Top Five Jekyll Plugin Gems (by GitHub Stars)

And the winner is…

  1. Assets 3 ★579 (gem: jekyll-assets) by Jordon Bedwell et al – asset pipelines
  2. Scholar ★328 (gem: jekyll-scholar) by Sylvester Keil et al – extensions for the blogging scholar (citations & bibliographies)
  3. Picture Tag ★296 (gem: jekyll-picture-tag) by Rob Wierzbowski et al – easy responsive images
  4. Sitemap Generator ★270 (gem: jekyll-sitemap) by Ben Balter et al – a plugin to silently generate a sitemaps.org compliant sitemap for your site
  5. Feed ★206 (gem: jekyll-feed) by Ben Balter et al – a plugin to generate an Atom (RSS-like) feed of your posts

More info @ Awesome Jekyll Plugins »
